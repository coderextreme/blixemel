# blixemel
Blender XML read and write scripts

instructions:

```
git clone https://github.com/coderexteme/blixemel
cd blixemel/scripts
bash export.sh
bash import.sh  # Check Action Editor under Dope Sheet in Animation (select other action)
bash exportgramps.sh
bash importgramps.sh # Check Action Editor under Dope Sheet in Animation (select other action)
```
Edit python scripts as desired.  Please try to repair armature import/export and extra action if you can.
